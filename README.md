### 项目简介：

本项目主要用于备份和分享自用图标库、分流规则及部分插件配置，适用于 Quantumult X、Loon、Surge、Fileball、Senplayer、Yamby、Hills 等工具。

图标内容包括常用应用图标、国家/地区图标、机场 Logo、Emby 相关图标等，后续会不定期补充和更新。
